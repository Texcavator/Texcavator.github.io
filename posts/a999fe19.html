数字三角形

题目在这里

题目

给定一个如下图所示的数字三角形，从顶部出发，在每一结点可以选择移动至其左下方的结点或移动至其右下方的结点，一直走到底层，要求找出一条路径，使路径上的数字的和最大。

                              7
                              3   8
                     	 8   1   0
                          2   7   4   4
                      4   5   2   6   5

输入格式

第一行包含整数 n，表示数字三角形的层数。

接下来 n 行，每行包含若干整数，其中第 i 行表示数字三角形第 i 层包含的整数。

输出格式

输出一个整数，表示最大的路径数字和。

数据范围

1 ≤ n ≤ 500,
−10000 ≤ 三角形中的整数 ≤ 10000

输入样例

输出样例

解法

每个点只可能从左上方点 or 右上方点走过来，所以在这两个地方取最大值加上当前点的权值即可

最长上升子序列

题目在这里

题目

给定一个长度为 N 的数列，求数值严格单调递增的子序列的长度最长是多少。

输入格式

第一行包含整数 N。

第二行包含 N 个整数，表示完整序列。

输出格式
输出一个整数，表示最大长度。

数据范围

1 ≤ N ≤ 1000
−109 ≤ 数列中的数 ≤ 109

输入样例

输出样例

解法

这道题的f[i]表示的是前 i 个字符中的最长上升子序列
在遍历第 i 个字符时，二重循环 1 - (i - 1) 个字符，f[i]即为f[i]和f[j] + 1的最大值

优化

两层循环O(n^2)可能一不小心就爆了
所以进行一下 ==二分== 优化

创建数组q[i]表示序列长度为 i 时的序列的最后一个数
对于每一个元素a[i]，在q[n]中进行二分查找，找到小于a[i]的最大的一个数q[j]，将a[i]接到这个子序列之后，修改q[j + 1]的值为a[i]，再更新一下长度的最大值

最长公共子序列

题目在这里

题目

给定两个长度分别为 N 和 M 的字符串 A 和 B，求既是 A 的子序列又是 B 的子序列的字符串长度最长是多少。

输入格式

第一行包含两个整数 N 和 M。

第二行包含一个长度为 N 的字符串，表示字符串 A。

第三行包含一个长度为 M 的字符串，表示字符串 B。

字符串均由小写字母构成。

输出格式

输出一个整数，表示最大长度。

数据范围

1 ≤ N, M ≤ 1000

输入样例

输出样例

解法

这道题的f[i][j]表示的是在 A 中的前 i 个字符和 B 中的前 j 个字符里选，最长公共子序列的长度是多少

当不选 A 中的最后一个字符 i 时，f[i][j] = f[i - 1][j]
当不选 B 中的最后一个字符 j 时，f[i][j] = f[i][j - 1]
先在这两个中取最大的

如果选最后一个字符，也就是 A 和 B 的最后一个字符相等的情况下
选取f[i][j]和f[i - 1][j - 1] + 1中最大的

最短编辑距离

题目在这里

题目

给定两个字符串 A 和 B，现在要将 A 经过若干操作变为 B，可进行的操作有：

  • 删除–将字符串 A 中的某个字符删除。
  • 插入–在字符串 A 的某个位置插入某个字符。
  • 替换–将字符串 A 中的某个字符替换为另一个字符。

现在请你求出，将 A 变为 B 至少需要进行多少次操作。

输入格式

第一行包含整数 n，表示字符串 A 的长度。

第二行包含一个长度为 n 的字符串 A。

第三行包含整数 m，表示字符串 B 的长度。

第四行包含一个长度为 m 的字符串 B。

字符串中均只包含大小写字母。

输出格式

输出一个整数，表示最少操作次数。

数据范围

1 ≤ n, m ≤ 1000

输入样例

输出样例

解法

这道题的f[i][j]表示把 A 的前 i 个字符换成 B 的前 j 个字符需要的步骤数

首先需要初始化f[0][i]和f[i][0]都等于 i（这个应该很好理解）
然后，最后一步一定是插入 or 删除 or 替换
插入：f[i - 1][j] + 1
删除：f[i][j - 1] + 1
替换：要看 A 的最后一个字符和 B 的最后一个字符是否相等

  • 相等的话：f[i - 1][j - 1]
  • 不相等的话：f[i - 1][j - 1] + 1
